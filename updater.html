使用须知

注意: • 需要先安装 Arcane 才可以运行, 联网并同步时间, 同步服务器. • 部分地区dns污染的, 还需要修改dns, 才可以正常下载 • 虚拟系统: 有联网保护, 无ban机风险 • 真实系统: 无联网保护, 有ban机风险

同步时间

  1. 打开DBI
  2. 进入工具菜单
    英文: Tools > NTP time sync
    俄语: Инструменты > Синхронизация времени по NTP
  3. 打开NTP联网同步工具
  4. 同步完成

升级安装

  • 按 L + ✚ 打开 Arcane
  • 按 ▶ 打开 Update
  • 打开 Proxy 开启 On (要求arcane 1.2.0+)
    • 在海外或者自己魔法的话, 请保持 Off

一键更新

  • Packages >Update All : 更新所有Packages和支持
  • AIO >Release : 升级 正式版整合包
  • AIO >Pre-Release : 升级 先行版整合包

单独安装/更新

  • HomeBrew Apps > ... : 自制应用
  • Overlays > ... : 菜单插件
  • Packages >... : 工具包及系统
  • Sys Modules >... : 系统模块
  • AIO >... : 整合包
  • Sync >... : 同步服务器
    • Install: 安装
    • Uninstall: 卸载
