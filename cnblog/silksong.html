nSwitch 丝之歌进度跟踪工具

导出NS存档

  • 通过 DBI 或者 JKSV 将 Hollow Knight Silksong 的存档复制到电脑上
  • 如果是zip格式, 先解压, 解压之后将 userX.dat 文件复制出来

NS存档转PC存档

  • 打开标题的网站(需要魔法)
  • 先勾选 Nintendo Switch Mode
  • 点击 Select File 选中上面备份出来的 userX.dat 文件
  • 转化完成之后, 找到右下角的 download encrypted (PC) 按钮
  • 打开并下载
