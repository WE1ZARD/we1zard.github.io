下载


  1. Lockpick_RCM.zip
  2. NxNandManager.zip (20系统需要用5.2.2版本, 从坚果云下载)
  3. Windows版本:NxWifiChanger.exe Python版本: NxWifiChanger-1.0.1.py (依赖库: ttkbootstrap )

导出 PRODINFO

提取密钥

  1. 下载Lockpick_RCM.zip解压后将bootloader覆盖到SD卡根目录
  2. 重启到Hekate主页, 选择Payloads > Lockpick_RCM.bin
  3. 用音量键控制上下, 关机键选择
  4. 密钥类型: Dump from SysNAND: 真实系统 密钥 Dump from EmuNAND: 虚拟系统 密钥
  5. 选好之后, 按关机键开始提取密钥prod.key, 保存位置在sdmc:\switch\prod.key
  6. 提取完成之后, 按关机键返回上一页, 然后选择Reboot to hekate重启
  7. 进入系统之后, 把刚才提取的密钥备份到电脑上, 所有需要用到密钥的工具都可以共用.

设置密钥

  • 下载NxNandManager.zip解压后用管理员身份打开 NxNandManager.exe
  • Options > Configure keyset
  • 选择 prod.key
  • 点击 Save保存

导出

  • switch关机之后将sd卡拔出, 用读卡器连接电脑
  • File > Open drive (文件系统的, 直接选择Open file)
  • 双击打开你的虚拟系统分区(通常是空间比较小的那个, 图2)
  • 如果你看不到隐藏分区, 就直接选择读卡器(图3 中的RW310 Lexar)
  • 选中 PRODINFO 右键 Decrypt & dump to file
  • 保存路径选择同文件夹即可
  • 导出完成之后, 点右上角的x关闭菜单
  • 手动备份一份原始未修改的PRODINFO.dec

Hekate 挂载办法

  • 打开 Hekate > Tools > USB Tools > Read-Only ON
  • 选择 emu RAW GPP 或者 eMMC RAW GPP (前面是虚拟系统, 后面是真实系统), 直接将系统挂载到电脑上, 其他操作一样.

修改 5G 信道

Hexit

展开 - 打开 [在线编辑工具](https://hexed.it/ '点击打开') - 将导出的 `PRODINFO.dec` 拖到编辑器标签栏 - 选中图中高亮位置的数值, 并将数值修改成同样的数值, 共两处 ![image](https://img2024.cnblogs.com/blog/3533742/202410/3533742-20241009201151905-846781761.png) - 点击左上角的 `Save as` 保存修改好的文件, 名字和原来一致`PRODINFO.dec`

NX Wifi Region Changer

- 下载NxWifiChanger.exe后用管理员身份运行 - 打开Open PRODINFO, 选择前面导出来的PRODINFO.dec - 打开下拉选项, 根据自己需求选择, 选中之后, 会自动生成.bak的备份文件,

导入 PRODINFO

  • 选择PRODINFO右键打开菜单, 选择Restore from file
  • 选择刚才修改好保存下来的PRODINFO.dec, 点 Yes, 开始导入修改后的参数
  • 导入完成之后, 点右上角的x关闭菜单
  • 然后点击左上角File > Close/drive 关闭文件和程序, 避免无法从电脑移除.
  • 重启NS连接wifi就可以正常连上5g wifi
