注意事项

  • 只适合未被打补丁的初代机器
  • 每次关机之后都需要重新注入
  • 第一次开启AutoRCM请确保你已经备份BOOT0和BOOT1 文件

注入器方法

  • 将已经安装好 整合包 的关机状态的switch的 右边 joycon滑轨 插入 短接器
  • 接入 注入器
  • 长按 音量键 + 同时按一下 开机键直到减号灯灭, 注入成功, 开机进入hekate
  • 卸下短接器, 拔掉注入器, 其他的操作就和硬破的switch一样了

电脑端方法

  • 下载 [TegraRcmGUI_vx.x_Installer_2.msi] 到电脑上
    • 网盘下载
    • 官网下载
  • 双击打开之后一直按 Next 直到完成安装
  • 打开桌面的 TegraRcmGUI 快捷方式
  • Settings > Install Driver 安装驱动 如果驱动有问题, 请手动安装 Zadig 打开下载好的 Zadig.exe 选择 libusbK (v3.1.0.0)
    点 Install Driver 开始安装驱动
    • 网盘下载
    • 官网下载
  • 打开 Inject payload 旁边的 文件夹图标
  • 选择电脑上 整合包 解压之后的 /Magic-Suite/payloader.bin
  • 将已经安装好整合包的关机状态的switch的 右边 joycon滑轨 插入 短接器
  • 长按 音量键 + 同时用usb线接入电脑
  • 正常情况下, 左下角图标会开始显示短接, 然后成功后变为绿色 RCM OK, 说明注入成功了
  • 注入成功之后, switch会开机进入Hekate菜单, 现在松开按键, 卸下短接器, 拔掉usb线, 其他的操作就和硬破的switch一样了

手机端方法

  • 下载 [5.3.Rekado.110.-release.apk] 并安装到安卓手机上
    • 网盘下载
    • 官网下载
  • 将 整合包 解压之后的 /Magic-Suite/payloader.bin复制进手机
  • 打开Rekado, 点击 + 选择刚才复制的 payloader.bin 的文件路径
  • 将已经安装好整合包的关机状态的switch插入 短接器
  • 长按 音量键 + 同时用usb线接入手机, 从手机弹出窗口, 选择payloader.bin注入
  • 注入成功之后, switch会开机进入Hekate菜单, 现在可以松开按键, 卸下短接器, 拔掉usb线, 其他的操作就和硬破的switch一样了

开启免短接 AutoRCM

优点:
- 开机不需要短接, 按开机键直接进入引导

缺点:
- 完全没电之后, RCM失效, 需要充20分钟左右 - 失效之后开机会进入黑屏RCM模式, 不是死机, 需要重新注入 - 每次升级系统之后, 需要重新打开AutoRCM

  1. 首次开启AutoRCM
    • 启动到Hekate, 打开 Hekate > Tools > Backup eMMC > eMMC BOOT0 & BOOT1, 进行备份(已备份直接步骤2)
  2. 打开 Hekate > Tools > Arch Bit • AutoRCM • Touch • Pkg1/2
    • 开启: AutoRCM On
    • 关闭: AutoRCM Off
