nSwitch 存档管理工具 - JKSV

JKSV是由JK开发的nintendo switch存档管理工具,
支持一键备份存档, 通过webDAV实现免会员云存档,
对拥有多部机器需要同步存档的朋友非常友好.

安装

NS安装

  • 联网并且同步时间和sync之后(如遇到无法成功下载, 请尝试改变DNS再试试)
  • 按住L+✚呼出Arcane菜单
  • 再按方向键右找到Updater打开HomeBrew Apps > JKSV > Install开始安装

电脑安装

  • 点击 下载 开始下载工具包
  • 下载后, 将zip解压内容switch覆盖到SD卡根目录

备份方法

设置黑名单

  • 移动到需要隐藏的游戏上按X打开设置, 选择黑名单即可
  • 添加之后, 将从一键备份里排除, 并隐藏.

一键备份

  • 按住R打开桌面任意程序进入相册工具或者打开hbmenu前端, 找到JKSV打开之后按Y即可
  • 存档生成保存在SD卡根目录/JKSV中

WebDAV云备份

用文本编辑软件打开sdmc: /config/JKSV/webdav_empty.json

以下以坚果云为例:

  1. 注册好账号, 登录之后, 点击右上角用户名, 进入账户信息
  2. 选择安全选项, 添加应用, 应用名称随便你填, 会生成对应的应用密码
  3. 将示例里的账户和刚生成的应用密码填写到以下对应的位置. (注意需要保留双引号)
{
  "origin": "https://dav.jianguoyun.com",
  "basepath": "dav",
  "username": "账户",
  "password": "应用密码"
}
  1. 完成填写之后将 webdav_empty.json 改成 webdav.json.
  2. 按住R打开桌面任意程序进入相册工具或者打开hbmenu, 再打开JKSV webDAV即可激活功能.

此为示范, 别把这部分复制进webdav.json

{
  "origin": "https://dav.jianguoyun.com",
  "basepath": "dav",
  "username": "xxxxxxxx@xxx.com",
  "password": "afadsdfjaaafaasldfjkals"
}
