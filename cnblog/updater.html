手动设置

DBI

  1. 打开DBI
  2. 进入工具菜单
    英文: Tools > NTP time sync
    俄语: Инструменты > Синхронизация времени по NTP
  3. 打开NTP联网同步工具
  4. 同步完成

升级安装

  • 按 L + ✚ 打开 Arcane
  • 按 ▶ 打开 Update
  • 打开 Proxy 开启 On (要求arcane 1.2.0+)
    • 在海外或者自己魔法的话, 请保持 Off

一键更新 - Packages >Update All : 更新所有Packages和支持 - AIO >Release : 升级 正式版整合包 - AIO >Pre-Release : 升级 先行版整合包

单独安装/更新 - HomeBrew Apps > ... : 自制应用 - Overlays > ... : 菜单插件 - Packages >... : 工具包及系统 - Sys Modules >... : 系统模块 - AIO >... : 整合包 - Sync >... : 同步服务器
• Install: 安装
• Uninstall: 卸载

常见问题

  1. 无法正常在线安装升级
    • 确保已正确同步联网时间
    • 查看 Updater > proxy 是否为 On
    • 复制 sdmc:/config/arcane/proxy.ini 文件到电脑上并用文本编辑工具打开 [config] proxy=https://gh.llkk.cc
    • 查看可用中转站
    • 将proxy=后面的网址替换成你电脑上显示可用的网址
    • 然后将编辑好的 proxy.ini 覆盖到 sdmc:/config/arcane/proxy.ini
    • 要求 arcane 1.2.0+
