「switchfin」switch上的Jellyfin视频播放器

关于

作者: dragonflylee 源地址: Switchfin

  • NS版本的Jellyfin, 可以通过网络播放自己媒体库, 支持硬解.

  • 下载地址: 坚果云

安装

  1. 解压switchfin.zip
  2. 填写按照自己搭建的媒体库要求进行填写/switch/SwitchFin/config.json
  3. 将switch文件夹复制到sd卡根目录

config.json填写示范(alist)

{
  "remotes": [
    {
      "name": "webdav",
      "url": "webdav://192.168.50.7:5244/dav",
      "user": "登录名",
      "passwd": "登陆密码"
    },
    {
      "name": "local",
      "url": "file://videos"
    }
  ]
}
