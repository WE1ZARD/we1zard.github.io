分区教程

用途

转换内存卡格式: exFAT -> FAT32 创建系统分区

注意事项

重新分区会格式化, 导致资料丢失 请提前做好资料备份!!!

内存卡分区

此操作会导致内存卡格式化 总文件大小 100 mb 以内 会自动备份 超过的需要手动备份!!!

  1. 重启, 按住 音量键减号 进入 hekate 主菜单
  2. 打开 hekate > Tools > Partition SD Card
  3. 点击后, 进入设置界面
  4. 设置好之后, 点 Next Step 重新分区

    初次转换格式需要先拉满绿条, 格式化之后, 退出页面再重新进, 才可以设置.

emuMMC (RAW)

  • 虚拟系统分区大小, 系统固定占用 3g
  • 其他空间可用于安装游戏或者游戏存档
  • 请确保分区大于官方系统已用的机身内存

Linux (EXT4)

  • Linux 系统分区
  • 最低要求: 12g

Android (USER)

  • 安卓系统分区
  • 最低要求: 8g
  • 分区格式:
    • Legacy 传统分区: 适用 android 10-11
    • Dynamic 动态分区: 适用 android 13+

eMMC 分区

此操作会导致机身内存的用户资料丢失 请提前备份, 避免丢失重要资料!!! 空间大小: OLED 56g, 其他 32g

  1. 重启, 按住 音量键减号 进入 hekate 主菜单
  2. 按住 hekate > Tools > Partition SD Card 直到跳转 eMMC Partition Manager
  3. 设置好之后, 点 Next Step 重新分区

eMMC (USER)

  • 除系统外的机身内存空间
  • 可用于安装游戏和存档
  • 最低要求: 2g

Linux (EXT4)

  • Linux 系统分区
  • 最低要求: 12g

Android (USER)

  • 安卓系统分区
  • 最低要求: 8g
    • Legacy 传统分区: 适用 android 10-11
    • Dynamic 动态分区: 适用 android 13+
