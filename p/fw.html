升级系统固件

注意事项

    为免重要资料丢失, 请先行做好备份, 再开始以下操作.
    请确保已经已经 制作虚拟系统
    只使用真实系统需要 设置启动项

文件操作

    下载解压 AIO.zip

  • 升级换包
    • 复制 bootloader, Magic-Suite 覆盖到 内存卡中
    将内存卡插入 switch
    开机之后, 稍等片刻程序自动处理完成, 自动进入系统, 操作完成!

    如果你之前没有制作虚拟系统, 自动重启之后会报错,
    按照 制作虚拟系统 操作即可.

  • 全新换卡
    • 打开 Magic-Suite 文件夹, 复制所有文件到 内存卡中,
    不要管外面单个的 bootloader 文件夹!!!
    复制完成之后, 开始 制作虚拟系统

    我要反馈

制作虚拟系统

    只玩真实系统可以跳过这步

    隐藏分区 速度快, 折腾不容易出问题 (★★★ 推荐 ★★★)
    文件格式 容易出问题, 部分游戏会卡顿 (不推荐)

隐藏分区系统

    重启按住 音量键减号 进入 Hekate

  • Hekate > Tools > Partition SD Card > OK
    • 此操作会导致内存卡格式化
    总文件大小不超过100mb会自动备份
    不满足条件的需要手动备份

  • Next Step
    • 点击之后, 开始格式化, 完成之后, 设置

    emuMMC (RAW)
    虚拟系统分区大小, 系统会占用3g
    剩余的空间将用于存档数据和游戏数据, 请根据自己情况做调整.
    请保证分区系统大于官方系统已用机身内存
    Linux (EXT4)
    Linux 系统分区大小, 最低要求: 16g
    Android (USER)
    安卓系统分区大小, 最低要求: 16g

    Next Step
    点击之后, 重新格式化并分区

  • Hekate > Create emuMMC > SD Partition
    • 此操作将创建一个官方系统的镜像到内存卡隐藏分区

文件格式系统

    重启按住 音量键减号 进入 Hekate

  • Hekate > Create emuMMC > SD File
    • 此操作将创建一个官方系统的镜像到内存卡

设置启动项

    用文本编辑工具打开 /Magic-Suite/bootloader/hekate_ipl.ini
    搜索 autoboot=1 并替换成 autoboot=2

    autoboot=1 虚拟系统
    autoboot=2 真实系统
