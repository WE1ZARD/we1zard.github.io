DBI 使用教程

主界面快捷键

  • X 键
    • 连接 MTP 外接设备
    可用于安装, 管理内存卡上的内容资料
    5号盘用于安装游戏

  • Y 键
    • 连接 USB 外接设备
    可用于安装外接设备上的内容资料

  • L 键
    • 浏览 已安装的程序

  • R 键
    • 浏览 已安装的自制软件

NTP 同步系统时间

用于联网服务
  • 主页面 > 工具 > NTP 时间同步
    • 看到绿色 OK 表示完成

密钥弹窗提示

功能受限, 缺少prod.keys, 并且每次打开都会提示
  • 导出密钥
    • 重启 switch, 按住 音量减号键 进入 hekate 主菜单
    打开 Payloads > Lockpick_RCM.bin 进入密钥提取工具
    用音量键控制上下, 用电源键打开选中项
    Dump from SysNAND -> 导出 真实系统密钥
    Dump from EmuNAND -> 导出 虚拟系统密钥

    完成后打开 Reboot to hekate 重启
    这时候再打开 DBI 主页面就不会再弹窗提示, 并且支持导出功能.

  • 密钥保存路径
    • /switch/prod.keys
    /switch/title.keys

SDK 导出与替换

用于强行将游戏要求的系统版本进行降级
但无法超过游戏功能要求的最低版本
  • 注意:
    • 需要提前准备 密钥

  • 导出 SDK
    • 主页面按 L 键 打开已安装的程序列表
    选中需要 SDK 版本的游戏按 + 键 导出SDK
    导出的 SDK 将按照版本号保存到路径 /switch/DBI/dumps/SDK/

  • 替换 SDK
    • 主页面按 L 键 打开已安装的程序列表
    选中需要替换 SDK 版本的游戏按 + 键 替换SDK
    替换前需将替换用的 SDK 保存到路径 /switch/DBI/dumps/SDK/
