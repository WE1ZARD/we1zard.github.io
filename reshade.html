ReShade 复古游戏滤镜插件

ReShade 是本人开发的一款switch上用的复古滤镜插件
支持自定义配置, 在我能力范围内尽可能的还原老游戏味道.

安装

  • 更新 Updater 和 Settings 至最新版本
  • 按 L + ✚ 打开菜单
  • 再按 ▶ 打开 Updater > Overlays > ReShade > Install 开始安装

Magic-Suite 最新版已内置, 无需额外安装

使用

  • 按 L + ✚ 打开菜单
  • 选择 ReShade > Scanlines 开启
  • 按住 左摇杆 + 右摇杆 退出
  • 快捷键 及 滤镜参数 可在 Settings > ReShade 进行修改设置
  • Reshade 中按 Y 可查看参数说明

    注意: 为保证流畅的游戏体验, 建议提高CPU的频率, 推荐CPU 2295+
