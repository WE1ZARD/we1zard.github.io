FPSLocker - 帧数锁定工具

FPSLocker 是masagrator开发的游戏增强工具之一,
依赖SaltyNX核心读取游戏的信息,
可以对 帧数设置, 缓冲设置, 刷新率设置 等进行独立设置.

下载

FPSLocker SaltyNX 依赖

额外功能

  • 解锁刷新率设置限制 LCD可用范围: 40-60hz(只有旧版的支持75hz) OLED可用范围: 45-60hz 移除Oled使用限制, 并增加安全范围 移除解锁声明
  • 下载解锁补丁: 无魔法要求

安装插件

  • 下载 Arcane, FPSLokcer, SaltyNX, 解锁补丁
  • 解压覆盖到根目录, 重启NS

同步时间

手动同步

设置 > 主机 > 日期时间

联网同步

  1. 打开DBI
  2. 进入工具菜单
  3. 打开NTP联网同步工具
  4. 同步完成

安装补丁

NS 安装

  • 时间同步之后
  • 按L+✚呼出Arcane
  • 按▶打开工具包, 选择Updater
  • 全部补丁: Overlays > FPSLocker > 60 FPS: Install
    单独补丁: 游戏中打开 FPSLocker > Advanced Settings > Check/download config file

补丁使用

开启补丁

  • 游戏中打开FPSLocker
  • 打开Advanced Settings高级配置
  • 打开Convert Config to patch file: 补丁转换成配置文件
  • 重启游戏解锁
  • 解锁之后, 需要根据自己情况调整目标帧数, 通常需要配合超频使用

卸载补丁

  1. 按L+加号呼出Arcane
  2. 按▶打开工具包, 选择Updater
  3. 卸载所有补丁 Overlays > FPSLocker > 60 FPS: Uninstall

刷新率设置

支持同步

  • 游戏中打开FPSLocker
  • 设置Display Settings > Display Sync: On
  • 调整FPS target

不支持同步

  • 游戏中打开FPSLocker
  • 设置Display Settings > Display Sync: Off
  • 调整Refressh Rate

底座设置

注意: 需要真实底座输出到屏幕上, 不能使用Reverse-RT底座插件替代

  • 打开FPSLokcer > Display Settings > Docked Settings
  • 选择第二个Refresh Rate Wizard刷新率设置助手
  • 打开之后按X开始测试, 期间会要求不同按键来启动测试, 照做就可以了
  • 做完测试之后, 就可以在Allowed Refresh Rates设置可选择的屏幕刷新率
  • 然后返回上一页打开Change Refresh Rate设置你想要使用的屏幕刷新率
  • Allow patches to force 60Hz: 启用时, 允许补丁强制使用60Hz刷新率
  • Use lowest refresh rate for unmatched FPS Targets: 启用时, 给无法匹配指定帧数的游戏使用最低的刷新率

注意事项

  • 配置文件转换成功之后, 需要重启游戏才能生效.
  • 不可用的刷新率可以通过同步刷新率进行重置
  • OLED屏幕刷新率支持不够完善, 不要强行操作, 以免损坏硬件. • 首次安装SaltyNX需要重启NS才能生效 • 部分游戏需要下载补丁文件后转换成配置文件才能解锁功能 • 下载功能需要同步时间
