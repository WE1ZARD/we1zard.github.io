适用于 Switch 上的 Linux, Lakka 和 Android 超频参数设置
配置文件路径 /bootloader/ini/*.ini

使用方法

下框会自动根据设置生成的配置参数,
请根据个人情况做相应的调整,
完成后将参数添加进对应的配置文件,
点击代码框复制到粘贴板.

ram_oc=0000000
ram_oc_opt=0

基础设置

Vdd2: 1050 mV
Vddq: 550 mV

初代
Vdd2: 1050-1237 mV
其他
Vdd2 1050-1175 mV
Vddq 550-650 mV




DVFS 设置

内存频率: 0
延迟: 0
核心时序: 0﹪
tREFI: 100﹪

tREFI 受温度影响! (初代机最多支持200%)

DVFS 高级设置

tRP: 0﹪
tRAS: 0﹪
tRCD: 0﹪
tRTW: 0﹪
tWTR: 0﹪
tRFC: 0﹪
tRRD/tFAW: 0﹪




tRTW 和 tWTR 大多数时候取决于内存频率, 在低频上通常可以多拉紧一点.
以上时序开启之后不再受 DVFS设置 中的核心时序影响, 以上未列出的时序仍受影响.
