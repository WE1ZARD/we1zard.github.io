用户指南

换包操作流程

注意事项

    为免重要资料丢失, 请先行做好备份, 再开始以下操作.
    请确保已经已经 制作虚拟系统
    只使用真实系统需要 设置启动项

文件操作

    AIO.zip

    下载 AIO.zip 解压
    复制 bootloader, Magic-Suite 覆盖到 内存卡中
    将内存卡插入 switch
    按电源键开机, 之后程序处理完之后会自动进入系统, 操作完成!

设置启动项

    用文本编辑工具打开 /Magic-Suite/bootloader/hekate_ipl.ini
    搜索 autoboot=1, 并替换成 autoboot=2

    autoboot=1 虚拟系统
    autoboot=2 真实系统

制作虚拟系统

    只玩真实系统可以跳过这步

    隐藏分区 速度快, 折腾不容易出问题 (★★★ 推荐 ★★★)
    文件格式 容易出问题, 部分游戏会卡顿 (不推荐)

隐藏分区系统

  • Hekate > Tools > Partition SD Card > OK

    • 此操作会导致内存卡格式化
    总文件大小不超过100mb会自动备份
    不满足条件的需要手动备份

  • Next Step
    • 点击之后, 开始格式化, 完成之后, 设置

    emuMMC (RAW)
    虚拟系统分区大小, 系统会占用3g
    剩余的空间将用于存档数据和游戏数据, 请根据自己情况做调整.
    请保证分区系统大于官方系统已用机身内存
    Linux (EXT4)
    Linux 系统分区大小, 最低要求: 16g
    Android (USER)
    安卓系统分区大小, 最低要求: 16g

    Next Step
    点击之后, 重新格式化并分区

  • Hekate > Create emuMMC > SD Partition
    • 此操作将创建一个官方系统的镜像到内存卡隐藏分区

文件格式系统

  • Hekate > Create emuMMC > SD File
    • 此操作将创建一个官方系统的镜像到内存卡
