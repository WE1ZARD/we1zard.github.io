注意事项

    重新创建 Android 系统分区需要重新分区格式化.
    请提前做好资料备份!!!

    需要提前 提取手柄蓝牙匹配数据 到内存卡上.

下载文件

    夸克网盘
    下载zip解压后, 文件复制进内存卡.
    再将 bootloader/ini/android.ini 覆盖到内存卡同路径上

准备工作

    重新分区
    分区创建完成之后, 打开 Hekate > Tools > USB Tools > SD Card 并用USB线连接电脑.

  • 复制系统文件
    • 将前面下载好的 bootloader 和 switchroot 和 *.zip 复制粘贴到内存卡上.
    复制完成之后, 从电脑上断开USB连接.

    如果你打算刷入 eMMC, 需要确保 android.ini 中有 emmc=1 这一行.
    如果你打算刷入 内存卡分区, 需要确保 android.ini 中没有 emmc=1 这一行.


刷入引导

  • 刷入内存卡分区
    • Hekate > Tools > (直接打开) Partition SD Card > SD Partition Manager

  • 刷入 eMMC 分区
    • Hekate > Tools > (按住跳转) Partition SD Card > eMMC Partition Manager

  • 确认操作
    • 点击底下的 Flash Android 开始刷入安卓系统引导
    两次弹窗提示都选 Continue, 有要求 Power 确认的都按电源键确认开始

    完成后自动进入安卓的 恢复模式 Recovery

初始化系统

  • 手动进入恢复模式
    • 如果上一步你没有自动进入恢复模式
    返回 Hekate 主菜单打开 More Configs
    按住音量减号键同时打开 LineageOS, 手动进入恢复模式.

  • 初始化数据
    • 点击 Factory reset 进入重置菜单
    Format data/factory reset > format data
    Format system partition > Yes

    完成之后点 <- 返回首页

安装系统

  • 选择安装文件
    • Apply update > choose from SWTICH SD
    选择前面复制到内存卡的zip文件

  • 跳过验证
    • 当你使用的是非官方前面的安装包, 安装过程出现签名验证失败

    Signature verification failed
    Install anyway?
    点击 yes

    完成之后点 <- 返回首页

  • 开启 ADB
    • Advanced > Enable ADB
    完成之后点 <- 返回首页

    点击 Reboot system now 重启进入安卓系统

解锁网络 - 有线版本

  • 开启 USB DEBUG 模式
    • Settings > About tablet > Build number
    连点7次本机版本号激活 开发模式

    System > Developer Options > USB debugging
    打开并点 OK 同意
    打开 Rooted debugging
    用 USB 线连接可以进行 ADB 操作的设备
    从屏幕上方下滑打开通知中心
    打开 Charging this device via USB
    选择 File Transfer

  • 解锁网络限制
    • 在安装了 ADB 工具的设备上打开终端, 依次输入以下命令:

    sudo adb shell settings delete global captive_portal_https_url

    sudo adb shell settings delete global captive_portal_http_url

    sudo adb shell settings put global captive_portal_https_url https://connect.rom.miui.com/generate_204

    sudo adb shell settings put global captive_portal_http_url http://connect.rom.miui.com/generate_204

    sudo adb shell settings put global time_zone Asia/Shanghai

    sudo adb shell settings put global ntp_server ntp1.aliyun.com

    正常情况下, 完成以上操作, 你的网络限制会解锁, 原本已经连接的网络只需要关闭重新打开即可刷新状态.

解锁网络 - 无线版本

  • 开启开发模式
    • Settings > About tablet > Build number
    连点7次本机版本号激活 开发模式

    System > Developer Options > Wireless debugging
    开启无线调试模式

  • 安装 ADB
    • 安卓手机, 安装 甲壳虫ADB助手

  • 解锁网络限制
    • 在安装了 ADB 工具的设备上打开终端, 依次输入以下命令:

    sudo adb shell settings delete global captive_portal_https_url

    sudo adb shell settings put global captive_portal_https_url https://connect.rom.miui.com/generate_204

    sudo adb shell settings put global captive_portal_http_url http://connect.rom.miui.com/generate_204

    sudo adb shell settings put global time_zone Asia/Shanghai
