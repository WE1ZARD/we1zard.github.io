用途

    转换内存卡格式: exFAT -> FAT32
    创建系统分区: 隐藏分区, Android, Linux

注意事项

    重新分区会格式化, 导致资料丢失,
    请提前做好资料备份!!!

内存卡分区

    重启按住 音量键减号 进入 Hekate

  • Hekate > Tools > Partition SD Card
    • 此操作会导致内存卡格式化
    总文件大小不超过100mb会自动备份
    不满足条件的需要手动备份

  • Next Step
    • 点击之后, 开始格式化, 完成之后, 设置

    emuMMC (RAW)
    虚拟系统分区大小, 系统会占用3g
    剩余的空间将用于存档数据和游戏数据, 请根据自己情况做调整.
    请保证分区系统大于官方系统已用机身内存

    Linux (EXT4)
    Linux 系统分区
    最低要求: 12G

    Android (USER)
    安卓系统分区
    最低要求: 8G
    - Legacy 传统分区: Android 10-11
    - Dynamic 动态分区: Android 13+

    Next Step
    点击之后, 重新格式化并分区


eMMC分区

    重启按住 音量键减号 进入 Hekate

  • Hekate > Tools
    • 按住 Partition SD Card 直到自动跳转 eMMC Partition Manager
    空间大小
    - OLED: 56G
    - 其他: 26G
  • 重新分区 eMMC
    • eMMC (USER)
    指除官方系统以外的内存空间
    可用于安装游戏和游戏存档
    最低要求: 2G

    Linux (EXT4)
    Linux 系统分区
    最低要求: 12G

    Android (USER)
    安卓系统分区
    最低要求: 8G
    - Legacy 传统分区: Android 10-11
    - Dynamic 动态分区: Android 13+

    Next Step
    点击之后, 重新格式化并分区
