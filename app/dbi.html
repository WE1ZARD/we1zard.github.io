主界面快捷键

    • 连接 MTP 外接设备
    可用于安装, 管理内存卡上的内容资料
    5号盘用于安装游戏

    • 连接 USB 外接设备
    可用于安装外接设备上的内容资料

    • 浏览 已安装的程序

    • 浏览 已安装的自制软件

NTP 同步系统时间

    用于联网服务

  • 主页面 > 工具 > NTP 时间同步
    • 看到绿色 OK 表示完成

密钥弹窗提示

    功能受限, 缺少prod.keys, 并且每次打开都会提示

    提取密钥

    完成之后, 再打开 DBI 主页面就不会再弹窗提示, 并且支持导出功能.

SDK 导出与替换

    用于强行将游戏要求的系统版本进行降级
    但无法超过游戏功能要求的最低版本

  • 注意:
    • 需要提前 提取密钥

  • 导出 SDK
    • 主页面按 打开已安装的程序列表
    选中需要 SDK 版本的游戏按 导出SDK
    导出的 SDK 将按照版本号保存到路径 /switch/DBI/dumps/SDK/

  • 替换 SDK
    • 主页面按 打开已安装的程序列表
    选中需要替换 SDK 版本的游戏按 替换SDK
    替换前需将替换用的 SDK 保存到路径 /switch/DBI/dumps/SDK/

  • 下载 SDK
    • 夸克网盘
    下载后复制到内存卡/switch/DBI/dumps/SDK/
