DBI 使用教程

免责声明

仅用于个人学习, 非商用, 侵删.

下载

直接下载

快捷键

  • X 连接 MTP 模式 外接设备
  • Y 连接 USB 模式 外接设备
  • L 浏览 已安装的程序
  • R 浏览 已安装的自制软件

同步时间

  • 用于下载服务, 不同步无法正常使用
  • 主页面 > 工具 > NTP 时间同步
  • 看到绿色 OK 表示完成

弹窗问题

  • 不影响安装, 但是所有需要密钥的高级功能都不能使用
  • 按照教程处理后, 不再弹窗, 并且解锁所有功能

提取密钥教程

SDK 使用

  • 所有操作都 需要密钥
  • 用于低版本系统运行高版本游戏
  • 仅限未使用高级系统版本功能的游戏

导出 SDK

  1. 主页面 按 L 打开已安装的程序列表
  2. 选中需要 SDK 版本的游戏按 按 加号键 导出 SDK
  3. 导出的 SDK 将按照版本号保存到路径 sdmc:/switch/DBI/dumps/SDK/

替换 SDK

  1. 主页面 按 L 打开已安装的程序列表
  2. 选中需要替换 SDK 版本的游戏 按 加号键 替换 SDK

    需先将 SDK 保存到路径 sdmc:/switch/DBI/dumps/SDK/
