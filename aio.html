安装流程 - Magic Suite

免责声明

仅用于个人学习, 非商用, 侵删.

注意事项

为免重要资料丢失, 请先行做好备份, 再开始以下操作.
请确保已经已经 制作虚拟系统
只使用真实系统需要 设置启动项

暂不支持 22.0.0

下载安装

直接下载 网盘下载

升级

  1. 下载解压后, 复制 bootloader, Magic-Suite 2个文件夹
  2. 覆盖到 sdmc:/ 内存卡根目录
  3. Switch 插入内存卡
  4. 开机后进入程序自动换包, 稍等片刻之后自动进入虚拟系统

    假如虚拟系统不存在, 重启后会报错, 按照 制作虚拟系统教程 操作即可

换卡

  1. 下载解压后, 进入 Magic-Suite 文件夹, 复制里面所有文件到 sdmc:/ 内存卡根目录
  2. 不要管外面单个的 bootloade` 文件夹!!!
  3. 复制完成之后, 开始 制作虚拟系统

制作虚拟系统

  • 分区系统: 速度快, 折腾不容易出问题 (推荐)
  • 文件系统: 容易出问题, 部分游戏会卡顿 (不建议)

制作分区系统

  1. 重启 按住 音量键减号 进入 hekate
  2. 按照 分区工具教程 创建分区
  3. 完成后, 返回 hekate 主菜单
  4. 打开右侧 Create emuMMC > SD Partition
  5. 此操作创建系统镜像到隐藏分区
  6. 内存卡中创建 sdmc:/emuMMC 文件夹, 未来安装的所有资料都存放在这里
  7. 完成后, 返回 hekate 主菜单, 打开 Launch > Atmosphere 进入虚拟系统

制作文件系统

  1. 重启 按住 音量键减号 进入 hekate
  2. 无需创建系统分区, 直接打开右侧 Create emuMMC > SD File
  3. 此操作创建系统镜像到 sdmc:/emuMMC 文件夹, 未来安装的所有资料都存放在这里
  4. 完成后, 返回 hekate 主菜单, 打开 Launch > Atmosphere 进入虚拟系统

离线升降级系统固件

  • 推荐 19.1.0 版本
  • 非必要不要升级, 尤其不要升级到 20.0.0+
  • 会导致很多游戏模组无法正常加载
  • 系统可用内存严重缩水
  • 加载速度明显变慢

离线升降级教程

在线更新包

  1. 按照教程同步时间之后
  2. 按 L + 加号 打开特斯拉菜单
  3. 按 方向右 打开 Updater > AIO > Pre-Release
  4. 下载完后会自动重启开始更新
  5. 更新完成后, 自动重启进入虚拟系统

系统备份教程

快捷键

特斯拉菜单:

  • 呼出: L + 加号

系统监控:

  • 退出: LS + RS
  • 临时隐藏: L + LS
  • 重置FPS统计: L + ZL

引导设置

编辑 sdmc:/bootloader/hekate_ipl.ini

  • autoboot=1: 自动引导虚拟系统
  • autoboot=2: 自动引导真实系统

系统支持

  • 4G/8G 切换 (仅限换了 8G 内存 查看方法)
  • 最新固件版本
  • 在线更新
  • 双虚拟系统引导
