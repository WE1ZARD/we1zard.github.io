介绍

基本命令

复制

cp /test1 /test2/

  • 复制 根目录 test1 文件夹 到 test2 文件夹中

cp /test1/ /test2/

  • 复制 根目录 test1 文件夹里的内容 到 test2 文件夹中

cp /test1/test3.ini /test2/

  • 复制 根目录 test1 文件夹里的test3.ini 到 test2 文件夹中

重命名

mv /test/test1.ini /test/test2.ini

  • 重命名 根目录 test 文件夹中的 test1.ini 为 test2.ini

移动

mv /test/ /

  • 移动 根目录 test 文件夹里的内容到 根目录

删除

del /test

  • 删除 根目录的 test 文件夹

del /test/

  • 删除 根目录的 test 文件夹里的内容

del /test/test.ini

  • 删除 根目录 test 文件夹里的 test.ini

下载

download https://test.com/test.zip /test.zip

  • 下载 https://test.com/test.zip 保存到根目录, 并命名为 test.zip

解压

unzip /test.zip /test/

  • 解压 根目录文件 test.zip 内容到 根目录 test 文件夹中
