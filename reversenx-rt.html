ReverseNX-RT 底座切换

介绍

依赖 SaltyNX 的特斯拉插件, 允许游戏运行时手动切换 掌机 / 底座 模式, 并输出对应的分辨率.

使用方法

Reset system control 恢复系统控制

  • 默认开启状态
  • 更改模式后, 切换成手动控制

Change mode 更改模式

  • 更改分辨率的输出模式
  • 强制系统按设置输出游戏分辨率

Save current settings 保存配置

  • 保存当前游戏的设置
  • 下次运行游戏自动加载设置
  • 配置保存路径 sdmc:/SaltySD/plugins/ReverseNX-RT/%016lX.dat
